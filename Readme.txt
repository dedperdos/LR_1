Що таке Git ? - це набір консольних утиліт для сп
В якому статусі можуть перебувати файли при роботі з Git ? -збережений змінений індексований
Для чого використовуеться команда add? -додае файл до репозиторію
Для чого використовуеться команда status? -перевіряє чи додані файли
Для чого використовується команда commit? -щоб зберегти зміни
Для чого використовуеться команда init? -ініціалізує Gіт у папці проекту
Що необхідно писати в коментарях комітів? -назву коміту
Як переглянути список комітів? -git log
Для чого використовується команда checkout? -дозволяє переміщатись між гілками
